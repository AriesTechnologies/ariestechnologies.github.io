ACFS Docs

Commands:

add() - Adds a file, folder, or drive to the filesystem

add_user() - Adds a user profile

attr() - Adds, changes, or displays file attributes

back() - Moves back from a file, folder, or drive

change_user() - changes user to control file permissions

delete() - Deletes a file, folder, or drive

del_user() - Deletes a user profile from the filesystem

dir() - Displays directories in the current folder

help() - Displays the list of commands

move() - Moves a file or folder to a new directory

open() - Opens a file, folder, or drive

quit() - Saves and exits the filesystem

rename() - Renames a file, folder, or drive

version() - Displays the version of the Filesystem

Command Documentation:

add(file, enter_new=False)
This command takes a file object, this object can be (File, Folder, SysFolder, Drive, Audio, or Picture)
add_user(name, admin=0)
This command takes a string that represents the name of the user and can take an integer that represents the access of the user
assoc(file_ext, assoc)
This command takes a string that represents the file extension and a string that represents the application association.
attr(name="", value="", filename="")
This command can take a string that represents the name of the attribute you want to change or add, secondly it can take a value that is associated with the name of the attribute, lastly it can take a filename which is the file you would like to add or change the attribute on. If none of these are put in the command then it will display the attribute list of that file.
back(index=-1)
This command can take an integer that represents how many directories it should go back.
change_user(name)
This command takes a string to represent the user folder.
delete(name)
The delete command takes a string that represents the name of the file you want to delete.
del_user(name)
This command takes a string that represents the name of the user you are deleting.
dir()
This command displays the files in a directory. You shouldn't need to use this since AtlasQDOS does this automatically.
help(cmd="")
This command can take a string that represents the command you want help with.
move(filename, directory)
This command takes a string that represents the name of the file that you would like to move, it also takes directory which represents the path to the new file location.
open(filename)
This command takes a string that represents the name of the file object you would like to open. If you are opening a file then filename takes a string that represents the type of opener used. The openers are w, a, r (writing, appending, reading).
quit()
This command saves and quits AtlasQDOS. This is important because if you don't do this before you close the program some changes may not be saved properly.
rename(filename, new)
This command takes a string that represents the name of the file you would like to change the name of. It also takes a string that represents the new name of the file.
version()
This command displays the version of AtlasQDOS
