Overview

SpartanOS is built to be quick, lightweight, easy-to-use. There are multiple versions of SpartanOS. Most are not able to be used in the real world since one was built using the Scratch coding language and one was corrupted. It is alright though, because there is one planned to be released in 2019! All our operating systems are usually created based off each other so you usually don't have to uninstall and reinstall the new OS. Instead, we release updates to the old operating system version to prepare it to switch over to the new version which is then reflected in the operating system settings, design, etc. The only time we wouldn't be able to do this is when we change the programming language or if the last version gets corrupted (Thanks AtlasOS Discovery :) ). In 2019 AtlasOS was renamed to SpartanOS, all operating systems before 2019 are still called AtlasOS.

Extensions

Regular File Extensions:

.udb (Universal Database File) = Used as a object file type (like a .json)

.sgf (Saved Game File) = Used for saving game information (can be encoded)

.cmn (Commission File) = Used for the Commission programming language

Multi-Use Extensions (Regular and SpartanOS):

.core (Core File) = Used as an important system file for SpartanOS (encoded)

SpartanOS File Extensions:

.drive (Drive File) = Used as the extension for removable storages in SpartanOS

.file (Basic File) = Used if the extension for a file in SpartanOS

.folder (Folder File) = Used as a folder in SpartanOS​​​​​​​

.scr (Shortcut Reference) = Used by SpartanOS as a shortcut reference

.mtf (Media Timestamp File) = Used for video/audio in SpartanOS

.psf (Pygame Surface File) = Used to images in SpartanOS

Operating Systems

SpartanOS 2024 (2022):

This version of SpartanOS is being created as we speak. Go to Updates for more information.

SpartanOS Eclipse (2021):

This version of SpartanOS is in Beta as we speak. As of now we are working on the API for SpartanOS applications. Go to Updates for more information.

SpartanOS Ambition (2019):

This version of SpartanOS has been released as the first downloadable version of our operating system. The picture at the top of the page is what SpartanOS Ambition looks like.

AtlasOS Discovery (-):

This version of AtlasOS was deprecated near the beginning of creation due to unforeseen problems with how the boot sector was created and how the security worked. We decided that it would be best to retire the version name as a tribute to the AtlasOS programming team for working so hard on this version throughout 2018 even though it was never released.

AtlasOS Powercell (2017):

This version of AtlasOS was the second version of the operating system. It was built using the Python programming language and ran using Windows 10.

AtlasOS Scratch (2016):

This version of the operating system is the first version of AtlasOS ever! It was built using the Scratch programming language. It looked a lot like Windows 10.
