World Creator

World Creator (2017): was AtlasStudios game where you could make your own world using a .udb file with certain letters/numbers/symbols to represent "blocks" of which the world was built using. It was coded in Python.

Universal Database File:

Our Universal Database File is a multi-use, multi-functional file type. We call it a universal file type because it can be used to create objects quickly and efficiently in almost any programming language. You can also use this type of file in programming languages to create classes, definitions, variables, etc. It is a lot like a .json and HTML.

Example:

<object : Model>

    <name : Dog>

</object>

<type : String>

    <name : str>

</type>

<object : 3d_Model>

    <name : House>

    <x : 100>

    <y : 100>

</object>

! { } =

{type = 3d}

    {__doc__ = Hello}

{/type}

(The exclamation point is used to change how the file is decoded. The characters after that represent the new syntax.)
